# 全身ROS2-Moveit2控制程序

## 分类:

vmr_vb2_jaka_k1: 装配JAKA-K1机械臂的整机全身控制包

vmr_vb2_taihu:装配钛虎机械臂的整机全身控制包



## 环境:

该部分运行在上半身的主控AGX Orin/Orin NX中，系统为Ubuntu22.04，使用ROS2版本为ROS Humble版本



## 说明：

该控制包内包含了整机的Moveit2-ROS2 Controller-ROS2 Control的整个控制链路，需要有一定Moveit2和ROS2的知识基础, Moveit2官网如下:
https://moveit.picknik.ai/main/index.html
